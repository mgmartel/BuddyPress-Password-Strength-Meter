Password strength meter for the 'change password' screen in BuddyPress

== Description ==

This plugin adds the WordPress password strength meter to the 'Change Password' field in the Profile->Settings screen in BuddyPress. 

Uses WordPress's own password strength meter to calculate password strength.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

== Frequently Asked Questions ==

= Why doesn't this plugin have any language files? =

This plugin uses strings in WordPress' language domain, and are translated along with WP.
